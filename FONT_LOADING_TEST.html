How to Test Font Loading

Using Chrome DevTools:

  1. Open DevTools (F12 or Ctrl+Shift+I)
  2. Go to Network tab
  3. Filter for "fonts" or "googleapis"
  4. Reload page (Ctrl+R)
  5. Look for requests to fonts.googleapis.com and fonts.gstatic.com
  6. Verify both return Status 200 OK
  7. Check waterfall timing (should be <200ms each)

Checking Font in Actual Use:

  1. On any portfolio page, right-click any text
  2. Select Inspect
  3. In DevTools, check the Styles tab
  4. Scroll down to find font-family declaration
  5. Should show: font-family: 'Urbanist', sans-serif
